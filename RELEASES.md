## 📋 Instruções de Instalação

### Windows (Recomendado)
1. Baixe `Mavit-1.0.0-Windows-FINAL.zip`
2. Extraia em qualquer pasta
3. Execute `Install.bat` como usuário normal
4. Atalho será criado na área de trabalho

### Linux
#### Método 1: DEB (Ubuntu/Debian)
```bash
wget https://github.com/maver/mavit/releases/download/v1.0.0/mavit_1.0.0_amd64.deb
sudo dpkg -i mavit_1.0.0_amd64.deb
```

#### Método 2: AppImage (Universal)
```bash
wget https://github.com/maver/mavit/releases/download/v1.0.0/Mavit-1.0.0.AppImage
chmod +x Mavit-1.0.0.AppImage
./Mavit-1.0.0.AppImage
```
